About Game Centre

Welcome to the Offline Game Centre 🎮
This project was built for fun and functionality — designed to work entirely offline, featuring a collection of classic games like Snake, Pong, and Tic-Tac-Toe.

It was built using Python, Tkinter, Pygame, Flask and SQLite — it’s a celebration of nostalgic arcade gaming brought to the modern day.

Created by: Daniel Ntuka
